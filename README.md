# **Digital-Voltmeter**
## **Description**
This program will emulate a `Digital Voltmeter` using the *MSP432* as the microcontroller unit (MCU). Additionally, the system uses the ADC peripheral with a 14-bit resolution to ensure precise readings. The `Digital Voltmeter` reads an analog input and converts into a voltage reading, which is displayed onto an *Liquid Crystal Display* (LCD). The software will be sampling every `100ms` (`10 Hz`) and update the LCD with a **98% accuracy**.

## **MSP432 LCD 16x02 Pinout**

| :---:       |  :--:      |   :---:      |
| `GND`, 10k Potentiometer Output (V&#177;) | 1| GND|  
| `VCC`, 10k Potentiometer Output (V&#177;) | 2| VCC|    
| 10k Potentiometer Output (V<sub>o</sub>) | 3| Contrast|    
| `P4.5` | 4| RS (Register Select): 0 – Command, 1 - Data|    
| `GND`| 5| R/W (Read/Write): 0 – Write, 1 - Read|    
| `4.4` | 6| Clock Enable|    
| `NC`| 7| Data 0|
| `NC`| 8| Data 1|    
| `NC` | 9| Data 2|    
| `NC`| 10| Data 3|    
| `P4.0` | 11|  Data 4|    
| `P4.1` | 12|  Data 5|    
| `P4.2` | 13|  Data 6|    
| `P4.3` | 14|  Data 7|    
| 100&#8486; to `VCC`| 15| Backlight Anode (+)|    
| `GND`| 16| Backlight Cathode (-)|
